# DoorsTower

Simple roguelike prototype made with Unity.

## Gameplay

Choose one of three doors.

Each door has different risks and rewards:

- Door 1: Safe
- Door 2: Balanced
- Door 3: Risky

Survive as many floors as possible.

## Current Features

- HP system
- Gold system
- Floor progression
- 3 doors with different probabilities
- Random treasure, heal and monster events
- Restart system

## Tech Stack

- Unity 6
- C#
- TextMeshPro
